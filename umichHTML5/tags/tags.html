Tags

Comment

The Comment Tag is used to add notes in the source code.

Example:

<!--This is a comment-->

Document Type

The Document Type Tag is used to inform the Client's browser that the document in use is an HTML document.

Example:

<!DOCTYPE html>

Hyperlink

The Anchor Tag is used to create a hyperlink to lead the client to another webpage.
Hyperlinks can either lead to a completely different document or a different location within the same document.
Important attributes include the Target attribute which determines whether a hyperlink should open on the current tab or a new one.

Example:

This hyperlink leads you to a collection of My UMICH HTML5 Projects (in a new tab)
This hyperlink leads you to the Top of this Document (in the same tab)
